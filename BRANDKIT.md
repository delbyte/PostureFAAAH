# 2. Logo System

## 2.1 Logo Philosophy

* Minimal
* Structured
* Geometric
* Strong negative space
* Feels engineered

## 2.2 Logo Construction

* Built on a strict grid system (8px baseline grid)
* Optical kerning refinement
* Consistent vertical alignment
* Tight letterspacing for authority

### Wordmark

Typeface: `"FrameGothic","FrameGothic Fallback"`

* Weight: 600 or 700
* Tracking: -2% to -3%
* Case: Sentence case or Title Case
* Avoid ALL CAPS unless for micro labels

* 18mm width minimum

# 3. Typography System

## 3.1 Primary Typeface

```
font-family: "FrameGothic","FrameGothic Fallback";
```

## 3.2 Font Weights

* 300 — Light (rare usage)
* 400 — Regular
* 500 — Medium
* 600 — Semibold
* 700 — Bold

---

## 3.3 Type Scale

Based on 8pt system

| Style      | Size | Weight | Line Height | Usage            |
| ---------- | ---- | ------ | ----------- | ---------------- |
| Display XL | 72px | 600    | 80px        | Hero sections    |
| Display L  | 56px | 600    | 64px        | Landing headings |
| H1         | 48px | 600    | 56px        | Section headers  |
| H2         | 36px | 600    | 44px        | Sub-sections     |
| H3         | 28px | 500    | 36px        | Content sections |
| H4         | 22px | 500    | 30px        | UI headers       |
| Body Large | 18px | 400    | 28px        | Main body        |
| Body       | 16px | 400    | 24px        | Default          |
| Small      | 14px | 400    | 20px        | Labels           |
| Micro      | 12px | 500    | 16px        | Metadata         |

---

## 3.4 Letterspacing Rules

* Large display text: -1% to -2%
* Body text: 0%
* Micro labels: +2% to +4% (uppercase)

---

## 3.5 Text Behavior

* Avoid decorative typography
* Never use italics excessively
* Emphasize using weight, not color
* Use sentence case primarily

# 5. UI & Interface Design Language

## 5.1 Layout System

* 8px grid system
* 1200–1440px max layout width
* 24px minimum padding
* 64px section spacing
* 16px internal component spacing

---

## 5.2 Card Design

* Background: #121420
* Border radius: 16px
* Border: 1px solid rgba(255,255,255,0.05)
* Soft inner shadow for depth

---

## 5.3 Buttons

### Primary Button

* Background: #4B5CFF
* Text: #FFFFFF
* Radius: 999px (pill style)
* Padding: 12px 24px
* Weight: 500
* Hover: #6E7CFF
* Active: Darkened 5%

### Secondary Button

* Transparent background
* Border: 1px solid #2A2E3B
* Text: #FFFFFF

---

## 5.4 Motion & Animation

* Duration: 200ms–300ms
* Easing: cubic-bezier(0.4, 0, 0.2, 1)
* Subtle scale (1.02 max)
* Soft opacity transitions
* Avoid bounce

Motion should feel:

* Smooth
* Controlled
* Professional
* Never playful
